# DFS

给定一个二叉树和一个目标和，判断该树中是否存在根节点到叶子节点的路径，这条路径上所有节点值相加等于目标和。

难点：

- 如何定义递归出口
- 递归中参数如何传递

`前序遍历 + 参数扩展`，向下递归的同时更新参数，当到达叶子节点或空节点时判断是否满足条件。
